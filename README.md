## 📜 설명
- Next.js와 Tailwind CSS로 만든 책 추천 사이트입니다.
- 사용자가 책 목록을 보고, 각 책의 상세 정보를 확인할 수 있게 해줍니다.

## ⚙️ 기능
- 📚 책 목록 보기
- 📖 책 상세 보기

## 🛠️ 기술 스택
- **프레임워크**: Next.js
- **스타일**: Tailwind CSS
- **언어**: TypeScript, JavaScript
